James – AI Virtual Assistant

James is a browser-based virtual assistant built using HTML, CSS, and JavaScript. It uses the Web Speech API to recognize voice commands and perform tasks such as searching the web and responding to user queries. The project demonstrates how JavaScript can be used to create interactive voice-controlled applications.
